# IT Service Desk Management System

## Objective
Develop a web-based IT Service Desk application to manage support requests efficiently.

## Users
1. Employee
2. IT Support
3. Admin

## Employee Features
- Register
- Login
- Logout
- Raise Ticket
- View Tickets

## IT Support Features
- View Tickets
- Assign Technician
- Update Ticket Status
- Resolve Tickets

## Admin Features
- User Management
- Reports
- Analytics

## Technology Stack
- Java
- JDBC
- MySQL
- JSP/Servlet
- GitHub
